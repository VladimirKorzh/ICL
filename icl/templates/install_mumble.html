Downloading and Installing Mumble

  • Mumble is an open source, low-latency, high quality voice chat software primarily intended for use while gaming.
  • It works similarly to RaidCall, Teamspeak or Ventrilo.
  • Client could be downloaded free of charge here (15mb).
  • During the install process simply click on next button, you don't need any special settings
  • After the install is complete, put a tick next to run mumble or run it manually afterwards.
  • During the first launch of mumble it will run a sound setup wizard, which will guide you in setting up sound.
  • If other users can't hear you in mumble, you should go to settings dialog. Place a tick next to 'Advanced' on the bottom of the screen and manually adjust your Audio Input.
